# Urban Bites ZW Self-Service POS System

## Overview
Urban Bites ZW is a touchscreen self-service Point of Sale system developed using Java Swing and MySQL.

The system allows customers to browse menu items, place orders, process payments and generate receipts without requiring direct assistance from a cashier.

## Features
- Touchscreen-style interface
- Product ordering
- Automatic tax calculation
- Cash, Card and EcoCash payments
- Receipt generation
- Sales recording
- Inventory management
- Database integration

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
